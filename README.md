# Wireworld Cellular Automata Experiments

## Quickstart

Clone the project

```sh
git clone https://github.com/djstompzone/wireworldca
cd wireworldca
```

Install with pip

```sh
pip install .
```

Run the module script

```sh
wireworldca
```

## Usage

For detailed usage instructions, run the module script with `--help`

```sh
wireworldca --help
```
